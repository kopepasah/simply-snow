A WordPress plugin that, when activated, will add a snowing effect on your site.

== Description ==
There are a lot of plugins that create a snowing effect on your site, but I wanted one that simply added a snowing effect without options, clutter or drastically affecting my sites speed.

No options.
No bloat.
Simply Snow.

== Installation ==
1. Upload \"simply-snow\" to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
